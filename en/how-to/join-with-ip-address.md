# How to Join a Server with IP Address

If you cannot find a server through normal search, you can join directly using an IP Address.

---

## Steps to Join

### Step 1: Open Neko Launcher
Launch the Neko Launcher application on your computer.

![Neko Launcher Step 1](https://cdn.furimoe.com/images/neko-launcher-step-1.png)

### Step 2: Login
Click the **Login** button to sign in to your account.

![Neko Launcher Step 2](https://cdn.furimoe.com/images/neko-launcher-step-2.png?dark=https://cdn.furimoe.com/images/neko-launcher-step-2-dark.png)

### Step 3: Go to Server Search Page
*Located in the top-right corner of the main window*

![Neko Launcher Step 3](https://cdn.furimoe.com/images/neko-launcher-step-3.png)

### Step 4: Enter IP Address
Enter the desired server's IP Address (e.g., `play.furi.moe`) into the search field.

![Neko Launcher Step 4](https://cdn.furimoe.com/images/neko-launcher-step-4.png)

### Step 5: Click Play
Click the **Join** or **Play** button to connect to the server.

![Neko Launcher Step 5](https://cdn.furimoe.com/images/neko-launcher-step-5.png)

---

## Example IP Address

```
play.furi.moe
```

---

## Troubleshooting

If you encounter errors:

- Make sure you are logged in
- Check your internet connection
- Try using a VPN if the server is in another region
- Contact the server administrator if the problem persists
